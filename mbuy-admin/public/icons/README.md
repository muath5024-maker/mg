# أيقونات التطبيق

يجب إضافة الأيقونات التالية لتطبيق PWA:

- `icon-192.png` - أيقونة بحجم 192×192 بكسل
- `icon-512.png` - أيقونة بحجم 512×512 بكسل

## كيفية إنشاء الأيقونات

1. قم بإنشاء شعار التطبيق بحجم 512×512 بكسل على الأقل
2. استخدم أداة مثل [RealFaviconGenerator](https://realfavicongenerator.net/) لإنشاء جميع الأحجام
3. ضع الملفات في هذا المجلد
